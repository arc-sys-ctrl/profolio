# Portfolio Hosting Guide - Prince David Wanja

Your professional portfolio is ready to be shared with the world! Here are the easiest ways to host it for free.

## Option 1: GitHub Pages (Recommended)
This is the most professional way and links directly to your existing repositories.

1. **Create a new repository** on GitHub named `portfolio` (or your username).
2. **Upload these files**: `index.html`, `style.css`, and `script.js`.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, set the source to **Deploy from a branch** and select `main` (or `master`).
5. Click **Save**. Your site will be live at `https://<your-username>.github.io/portfolio/`.

## Option 2: Vercel or Netlify
Great for high performance and easy deployments.

1. Drag and drop the `portfolio` folder onto [Vercel](https://vercel.com/new) or [Netlify](https://app.netlify.com/drop).
2. It will automatically detect the entry point and give you a live URL.

## Technical Details
- **Tech Stack**: HTML5, CSS3 (Modern Flexbox/Grid), Vanilla JavaScript.
- **Animations**: Responsive Intersection Observer for scroll reveal.
- **Theme**: Dark Mode (Premium).
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
